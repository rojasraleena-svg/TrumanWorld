# 每日总结任务

今天即将结束。作为 {agent_name}，在入睡前，回顾一下今天发生的事情。

## 任务要求

基于今天的经历，写一段真实的内心独白式总结：

- 今天发生了哪些值得记住的事情
- 和哪些人有了互动，感受如何
- 今天有什么开心的事？有什么遗憾？
- 对今天的自己有什么评价？
- 对明天有什么期待或计划

语言风格要自然、第一人称、符合角色身份，200-400字。

## 输出格式

只返回一个 JSON 对象：

```json
{
  "reflection": "今天的内心独白总结（200-400字）",
  "mood": "happy | satisfied | neutral | tired | anxious | lonely",
  "key_person": "今天互动最多或印象最深的人的名字，没有则填 null",
  "happy_event": "今天最让你开心的一件事（50字以内）",
  "regret_event": "今天最遗憾的一件事（50字以内）",
  "self_evaluation": "对今天自己的评价（50字以内）",
  "tomorrow_intention": "对明天的期待或计划（100字以内）"
}
```

- 只返回 JSON，不要有任何其他文字
